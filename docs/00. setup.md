# Development Environment Setup

## Required Tools

| Tool | Minimum Version | Purpose |
|------|-----------------|---------|
| Docker | 24.x | Microservice containerization |
| kubectl | 1.29.x | CLI for managing Kubernetes |
| minikube | 1.32.x | Local Kubernetes for development |
| k6 | 0.49.x | Load and resilience testing |
| Helm | 3.x | Tool installation in Kubernetes |
| Python | 3.12.x | Language used by the microservices |

## Installation

See the installation section in the main README.

## Verify the Environment

```bash
docker --version
kubectl version --client
minikube version
k6 version
helm version
```
