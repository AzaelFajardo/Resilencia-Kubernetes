
# Configuración del Entorno de Desarrollo

## Herramientas requeridas

| Herramienta | Versión mínima | Propósito |

|-------------|---------------|-----------|

| Docker      | 24.x          | Contenerización de microservicios |

| kubectl     | 1.29.x        | CLI para administrar Kubernetes |

| minikube    | 1.32.x        | Kubernetes local para desarrollo |

| k6          | 0.49.x        | Pruebas de carga y resiliencia |

| Helm        | 3.x           | Instalación de herramientas en K8s |

| Python      | 3.12.x        | Lenguaje de los microservicios |

## Instalación

Ver sección de instalación en el README principal.

## Verificar entorno

```bash

docker --version

kubectl version --client

minikube version

k6 version

helm version

```
